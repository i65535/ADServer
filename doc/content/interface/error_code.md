#### 权限认证错误码：

* **10101** 无效的token 
* **10102** 登录超时 
* **10103** 用户未登录 
* **10104** 用户token不存在 
* **10105** token 不完整 
* **10106** 时间偏移超过限制范围 
* **10107** 时间戳重复，调用太频繁或非法调用  
* **10108** 用户权限不够 
* **10109** 未定义的用户角色
* **10110** 分组不存在 
* **10111** 用户不存在 
* **10112** 密码错误 
* **10113** 密钥未通过验证，用户被踢出
